### @public / @protected / @private / @ignore
### @param
### @returns
### @var

## Format
### Javascript
### Macro

### SCSS
#### Variables
Needs at least a `@type` or a `@kind variable`.

```
/**
 * @type {Color}
 */
$color-green: green;
```

#### Classes
Needs at least a `@class` or a `@kind class`.

```
/**
 * @class
 */
.color-green { color: green; }
```
